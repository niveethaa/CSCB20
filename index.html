Course Description

This course provides students with a detailed and practical description of the various databases and web applications. Course material covers SQL, Relational Algebra, HTML, CSS and Python Flask. In addition, this course may not be taken with any other -C or -D level CSC course. Assesments through the course consists of assignments, midterm and a final exam at the end of the course. Over the course of the semester, the students will learn the following material in an interactive and practical manner. All must students must be able to write the final exam in-person and deferred exams and midterms must have a valid reason with a doctor's note.

Important Topics

Topic 1: Databases

  • Terminology and applications
  • Creating query and updating databases
  • The entity-relationship model design
Topic 2: Web Documents and Applications
  • Static and interactive Documents
  • Web servers and dynamic server-generated content
  • Web application development and interface with data bases
